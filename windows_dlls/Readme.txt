What is it?
M3C2.exe is an executable C++ program with the goal of detecting the diffference of point clouds between two time epochs. 
It is also easy to extend the program to change detection for multiple epochs.

How to use it?
In cmd, type the command line by following the format:
'M3C2 the-name-of-the-first-point-cloud the-name-of-the-second-point-cloud the-name-of-the-result-file'

For example, type the command below: 
M3C2 F5_March_2019_1cm_crop_crop_small_small.las F5_October_2019_1cm_crop_crop_register_small_small.las M3C2results.las
